# Frontend Practice Projects

A collection of small frontend projects I’m building while learning frontend development on my own.

These projects are simple, beginner-friendly experiments focused on improving my HTML, CSS, and JavaScript skills.


## Purpose

The goal of this repository is to:

- Practice frontend fundamentals
- Build small projects consistently
- Improve problem-solving with JavaScript
- Learn by doing
- Track my progress over time

## Tech Stack

Most projects will use:

- HTML
- CSS
- JavaScript

Some future projects may include:

- React
- Tailwind CSS
- APIs
